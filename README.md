# hijri-native

React Native TurboModule for Hijri (Islamic Umm al-Qura) calendar conversion using native OS APIs

## Installation


```sh
npm install hijri-native
```


## Usage


```js
import { multiply } from 'hijri-native';

// ...

const result = multiply(3, 7);
```
